📖 Why This Exists
Most "prompt collections" online are unstructured dumps. This library is built like a real engineering project: consistent naming, metadata for every prompt, and folders organized around how prompts are actually used day to day — not just theory.

🚀 How to Use
Browse a category folder relevant to your task.
Open the prompt `.md` file — each includes purpose, variables, and example output.
Copy the prompt, fill in any `{{variables}}`, and paste into your AI tool of choice.
New prompt? Use `PROMPT_TEMPLATE.md`.
🏷️ Metadata & Tags
Every prompt file includes YAML frontmatter (title, category, tool, tags, difficulty, version) for search and filtering. See `TAGGING_SYSTEM.md`.
